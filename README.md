# Lore Meter - SillyTavern Extension

AI 기반 캐릭터 분석 기능을 제공하는 SillyTavern 확장 프로그램입니다.

## 주요 기능

### 캐릭터 통계 및 분석

캐릭터 메뉴에서 하트 아이콘(💗)을 클릭하면 다음과 같은 기능을 사용할 수 있습니다:

- **캐릭터 통계**: 첫 만남 날짜, 함께한 시간, 총 채팅 수, 총 메시지 수
- **AI 캐릭터 분석**: 여러 채팅을 분석하여 캐릭터와 스토리에 대한 종합적인 리포트 생성

### AI 분석 기능

두 가지 분석 모드를 제공합니다:

1. **1인칭 분석**: 캐릭터의 시점에서 자신의 경험을 회고하는 형식
2. **3인칭 분석**: 전문가 관점에서 캐릭터와 스토리를 객관적으로 분석하는 형식

#### 분석 옵션

- **자동 분석**: 모든 채팅을 자동으로 선택하여 분석
- **직접 선택**: 원하는 채팅만 선택하여 분석
- **설정**: 분석 모드 변경 및 프롬프트 커스터마이징

#### 지원 모델

- gemini-2.5-flash-lite
- gemini-2.5-flash
- gemini-3-flash-preview

### 분석 결과 항목

#### 3인칭 분석
1. 주인공 분석 - 캐릭터의 성격, 특성, 변화
2. 상대 캐릭터 분석 - 주요 등장인물과 관계
3. 시나리오 및 플롯 - 스토리 전개와 테마
4. 로맨스 및 성적 요소 - 관계의 발전과 친밀도

#### 1인칭 분석
1. 나 자신에 대해 - 캐릭터의 자기 인식
2. 파트너들에 대해 - 상대방에 대한 감상
3. 순간들에 대해 - 인상 깊었던 장면들
4. 이야기들에 대해 - 전체적인 평가와 여운

## 설치

SillyTavern의 Extension Manager에서 이 폴더를 extension으로 등록하여 사용할 수 있습니다.

## 사용 방법

1. 캐릭터를 선택한 상태에서 캐릭터 메뉴를 엽니다
2. 상단에 표시되는 하트 아이콘(💗)을 클릭합니다
3. AI 모델을 선택합니다
4. "자동 분석" 또는 "직접 선택" 버튼을 클릭합니다
5. 분석 결과가 표시되면, 더블클릭하여 전체 화면으로 볼 수 있습니다

## 필요 사항

- SillyTavern
- Google AI Studio API 키 (Gemini 모델 사용을 위해 필요)
